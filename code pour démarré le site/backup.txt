Backup manuel (safe, avec demande de mot de passe) Copie/colle dans PowerShell :

$pgDump = "C:\Program Files\PostgreSQL\18\bin\pg_dump.exe"
$ts = Get-Date -Format "yyyy-MM-dd_HH-mm-ss"
$backup = "$env:USERPROFILE\Desktop\revolution_backup_$ts.dump"

$env:PGSSLMODE = "require"

$sec = Read-Host "Postgres password" -AsSecureString
$bstr = [Runtime.InteropServices.Marshal]::SecureStringToBSTR($sec)
try {
  $env:PGPASSWORD = [Runtime.InteropServices.Marshal]::PtrToStringBSTR($bstr)

  & $pgDump -h dpg-d6qvkh3uibrs739i3hkg-a.frankfurt-postgres.render.com `
    -U postgre_sql_rouk_user -d postgre_sql_rouk `
    --format=custom --no-owner --no-acl `
    --file "$backup"
} finally {
  Remove-Item Env:PGPASSWORD -ErrorAction SilentlyContinue
  [Runtime.InteropServices.Marshal]::ZeroFreeBSTR($bstr)
}

Get-Item "$backup" | Select-Object FullName, Length
```


- Il va te demander : Password: → colle le mot de passe → Entrée.
- À la fin, tu dois voir le fichier .dump sur ton Bureau + sa taille.
